Runtime Data Flow -- 运行时数据流

sequenceDiagram
    participant User as 研究人员
    participant CLI as CLI main
    participant Runner as BenchmarkRunner
    participant TM as TaskManager
    participant LA as LocalAccessor
    participant SP as Splitter
    participant MR as model_registry
    participant Model as PerturbationModel
    participant EV as Evaluator
    participant MetR as MetricRegistry
    participant Func as 指标函数
    participant ER as EvalResult

    Note over User, CLI: 阶段一: 命令解析
    User->>CLI: perteval run --benchmark norman19 --model mean_control
    CLI->>Runner: 创建 BenchmarkRunner

    Note over Runner, SP: 阶段二: 数据加载与划分
    Runner->>TM: get norman19
    TM-->>Runner: BenchmarkConfig YAML 配置
    Runner->>LA: load norman19.h5ad
    LA-->>Runner: AnnData 表达矩阵
    Runner->>SP: split transfer 方法
    SP-->>Runner: train / val / test 子集

    Note over Runner, Model: 阶段三: 模型训练与预测
    Runner->>MR: get mean_control
    MR-->>Runner: MeanControl 类 惰性解析
    Runner->>Model: train 训练集
    Model-->>Runner: 模型就绪
    Runner->>Model: predict 控制组细胞 + 扰动列表
    Model-->>Runner: 预测 AnnData

    Note over Runner, ER: 阶段四: 评估与输出
    Runner->>EV: evaluate PerturbationData
    loop 逐扰动计算
        EV->>MetR: get 指标名
        MetR-->>EV: MetricInfo 含函数引用
        EV->>Func: 调用 pearson_delta / mse / overlap_at_k
        Func-->>EV: 指标分数
    end
    EV->>ER: 构建 per_perturbation + aggregated
    ER-->>Runner: EvalResult
    Runner->>ER: to_json + to_csv
    ER-->>User: JSON / CSV 结果文件
